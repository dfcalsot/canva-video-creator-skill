# 🎬 Canva Video Creator Skill
### Para el Álbum Educativo **Luka & Dana**

> Skill completa para producir 12 videos musicales educativos en **Canva Pro**,
> combinando audio generado en Suno AI (West Coast + Gorillaz, 432 Hz) con
> animaciones, personajes ilustrados e IA generativa de imágenes.

---

## 🌟 ¿Qué hace esta skill?

Guía paso a paso para convertir cada canción del álbum en un video completo de
2:30–3:00 minutos, listo para YouTube, Instagram o uso educativo en el aula.

```
Audio MP3 (Suno)  +  Imágenes personajes (Firefly/Ideogram)  +  Letra
         ↓
    Canva Pro (diseño + animación + sincronización)
         ↓
    Video MP4 1080p — listo para publicar
```

## 🚀 Cómo usar esta skill

### Requisitos
- ✅ Canva Pro (canva.com)
- ✅ Canciones generadas en [Suno AI](https://suno.com) (MP3 descargados)
- ✅ Cuenta gratuita en [Adobe Firefly](https://firefly.adobe.com) o [Ideogram](https://ideogram.ai)

### Flujo en 8 pasos

```
1. PREPARAR   → Descargar MP3s de Suno + generar imágenes de personajes
2. PLANTILLA  → Abrir Canva Pro, crear diseño Video 1920×1080
3. DISEÑAR    → 12 slides por canción con letra, personajes y fondo
4. ANIMAR     → Texto con Fade/Pop, elementos con Beat Sync
5. AUDIO      → Subir MP3 → arrastrar al timeline → sincronizar
6. EXPORTAR   → Compartir → MP4 → 1080p → Incluir audio
7. DUPLICAR   → Usar la plantilla del Track 01 para los demás 11
8. PUBLICAR   → YouTube / Instagram / uso educativo
```

### Tiempo estimado
| Tarea | Tiempo |
|-------|--------|
| Generar imágenes de personajes (x8) | 30–40 min |
| Diseñar plantilla Track 01 | 45–60 min |
| Cada canción adicional (x11) | 15–20 min c/u |
| **Total álbum completo** | **~5–6 horas** |

```

□ Slide 5: Pre-coro (transición visual)
□ Slides 6–7: Coro (texto grande, colores vibrantes)
□ Slides 8–9: Verso 2 + Bridge
□ Slides 10–11: Coro final + Outro
□ Slide 12: Créditos

AUDIO Y EXPORT
□ MP3 subido a Canva
□ Audio sincronizado con Beat Sync o manualmente
□ Duración total: 2:30–3:00 min ✓
□ Exportado en MP4 1080p con audio incluido
□ Archivo nombrado: 01_El_Portal_del_Barrio_VIDEO.mp4
```

## 💡 Tips clave

- **Plantilla maestra:** Diseña solo el Track 01 completo, luego duplica para los 11 restantes. Cambia solo el fondo, letra, audio y personaje protagonista.
- **Beat Sync:** Selecciona todos los slides → clic derecho → "Sincronizar con música" para que Canva ajuste automáticamente los cortes al ritmo.
- **Dana se pronuncia DA·na** — siempre con acento visual en el texto si es posible.
- **Fondo IA:** Usa Canva Magic Media (Apps → Magic Media) para generar fondos únicos por canción sin salir de Canva.
- **Export con audio:** Asegúrate de activar "Incluir audio" antes de descargar.
